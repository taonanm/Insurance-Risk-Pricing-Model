# Insurance-Risk-Pricing-Model
This project models insurance claim severity and simulates actuarial pricing strategies using machine learning.
## Key Features
- Predicts insurance loss using Random Forest
- Applies log transformation to handle skewed data
- Segments policyholders into risk tiers
- Simulates premium pricing based on risk

## Tools Used
- Python (Pandas, NumPy, Scikit-learn, Matplotlib)
- Jupyter Notebook

## Results
- Model achieved an R² of ~0.47
- Successfully identified high-risk groups with higher average losses
